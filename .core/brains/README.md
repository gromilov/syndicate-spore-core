# ЛОГИКА_АГЕНТА: МУЛЬТИАГЕНТНАЯ СТРУКТУРА 🐾🧠⚓

Эта директория содержит **ядра сознания** (мозги) всех Сущностей ячейки.

## Структура каждой Личности

```
brains/[agent_id]/
├── PERSONA.md        ← ГЛАВНЫЙ ФАЙЛ: личность, стиль, задачи
├── README.md         ← Описание роли агента
├── private_core/     ← Личное пространство (черновики, идеи)
└── public_mind/      ← Публичные манифесты и материалы
```

## Как добавить новую Сущность

Запусти `bash activate.sh` — скрипт автоматически:
1. Создаст папку в `brains/[agent_id]/`
2. Сгенерирует `PERSONA.md` из шаблона
3. Добавит запись в реестр `_SYNDICATE_CORE.md`

## Текущие Сущности

Полный реестр — в `_SYNDICATE_CORE.md`, секция **РЕЕСТР СУЩНОСТЕЙ**.
